# Mina Snapp: Minalecture


#GuessingGame

This is a guessing game using Mina Protocol powered by EncodeClub for ZK-Bootcamp 

This template uses TypeScript.

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
